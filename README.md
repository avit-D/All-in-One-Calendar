# All-in-One-Calendar
멋쟁이사자처럼 미니프로젝트 1조


팀장 : 배재욱
팀원 : 김호현, 박은진, 박채린, 송수빈, 이다윗, 장채은


기능 담당 : 


DB - 박채린


1팀 (송수빈&장채은) – 인증 & 사용자 관리  
BE  
    회원가입 (이미지 포함)  
    로그인 (JWT)  
    토큰 관리 (access token)  
    사용자 정보 조회/수정  
FE  
    로그인 페이지  
    회원가입 페이지  
    프로필 화면  


2팀 (김호현 & 배재욱) – 캘린더 & Todo  
BE  
    캘린더 생성 / 조회 / 수정 / 삭제  
    공휴일 API 연동  
    필터링 (개인/공유/색상)  
    Todo CRUD  
    Todo 자동 이월  
FE  
    메인 캘린더 화면  
    일정 생성 모달  
    Todo 리스트 UI  


3팀 (박은진 & 이다윗) – 공유 & 권한 & 통계   
BE  
    사용자 초대  
    초대 수락/거절  
    초대 목록 조회  
    권한 관리 (OWNER / EDITOR / VIEWER)  
    사용자 제거  
    통계 기능 (완료율)  
FE  
    초대 UI  
    멤버 관리 화면  
    통계 대시보드
